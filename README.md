## About

**Efficacy of treatment based on ACT therapy in children under five years old with uncomplicated malaria in Africa**

This project aims to determine the efficacy of artemisinin-based combination therapy (ACT) to treat children under five with 
hyperparasitaemia in Sub-Saharan African countries using IPD from IDDO's studies repository. This is part of the TDR/WHO fellowship project.
